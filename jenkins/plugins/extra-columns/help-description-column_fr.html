Cette colonne permet d'afficher la description du job (au format HTML).

Il est aussi possible d'afficher le nom du job au dessus de la description en cochant "Afficher le nom du job".
Il est possible de rogner la description à un nombre défini de lignes en cochant "Rogner la description". Dans ce cas, il faut définir la quantité de lignes souhaitées.
Le découpage est fait sur les tags <br/>.
